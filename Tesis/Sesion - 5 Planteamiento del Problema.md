## Naturaleza epistemológica del problema
Un problema científico es una ==dificultad cognoscitiva== que no puede resolverse automáticamente con los conocimientos disponibles .
Las preguntas mas importantes afirman el planteamiento. 
	1. ¿ que fenómeno se observa ? 
	2. ¿por que constituye un problema ?
	3. ¿Qué consecuencias tiene si no se aborda ? 
## Identificación  del problema especifico 
Para poder identificar un problema de investigación genuinamente se sigue estos pasos : 
1. Que pueda formularse como pregunta interrogativa. 
2. Que admita respuesta empírica teórica 
3. Que sus variables sean observables o medibles 

> [!NOTE]
> Vacío del Conocimiento : Es cuando todavía no se a estudiado lo suficiente o no sea respondido todavía.   

## Contextualización 
Un proyecto de tesis tiene que tener si o si una contextualización si no lo tiene es retorica vacía no sirve en otras palabras. 
Existe varias niveles de contextualización las cuales son las siguientes: 

| Nivel         | Funcion                                                            |
| ------------- | ------------------------------------------------------------------ |
| Internacional | Relevancia global del problema                                     |
| Nacional      | Evidencia de pertinencia al contexto nacional en este caso peruano |
| Regional      | El problema se sitúa en el región de Ayacucho                      |
| Local         | Define la unidad concreta de análisis y la magnitud del problema   |
## Delimitación del problema de investigación 
La delimitación de la tesis es muy importante porque nos ayuda a definir con claridad las ==fronteras de estudio  o alcance.==
Existen varios tipos de delimitación y son los siguientes : 

| Tipo       | Pregunta clave                              |
| ---------- | ------------------------------------------- |
| Espacial   | ¿Dónde se realiza el estudio ?              |
| Temporal   | ¿ En que periodo se desarrolla ?            |
| Social     | ¿Quiénes son los objetos de estudio ?       |
| Conseptual | ¿ Que variables y constructos se estudian ? |
| Muestral   | ¿ A que población y muestra se aplicara ?   |
## Estructura del planteamiento
El planteamiento del problema puede estar estructurado en tres párrafos : 
1. Diagnostico : Describimos la situación actual con datos reales o actuales
2. Pronostico :  Este párrafo básicamente responde a esta pregunta ¿ Que pasara si no se soluciona o responde o actúa ? 
3. Control del pronostico: Este párrafo responde a la pregunta de ¿ Que se propone hacer para evitar el escenario pronosticado ?  ==Importante :  No es la solución implementada== 
## Justificación de la Investigación 
La justificación  argumenta  los aportes específicos del estudio  
Existe Dimensiones de justificación y son los siguientes : 

| Dimencion    | Pregunta                                                     |
| ------------ | ------------------------------------------------------------ |
| Teórica      | ¿ Que aporta al cuerpo de conocimiento de la disciplina ?    |
| Metodológica | ¿ Que innovación instrumental o de diseño ?                  |
| Practica     | ¿ Que problema concreto contribuye a resolver ?              |
| Social       | ¿Qué beneficios  genera para grupos sociales o comunidades ? |
