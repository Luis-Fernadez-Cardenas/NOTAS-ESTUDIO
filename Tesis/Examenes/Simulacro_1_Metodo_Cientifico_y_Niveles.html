Simulacro N°1 — Ciencia, Método y Diseños

Tesis I - Serie 500 - UNSCH | Analítico
1. Un investigador observa que en una empresa los empleados están renunciando masivamente. Decide aplicar una encuesta rápida a 50 exempleados solo para conocer la frecuencia de motivos generales, sin cruzar datos. ¿En qué nivel de investigación se encuentra este estudio?
Respuesta correcta: c). El nivel exploratorio responde a "¿Cuántos?" usando frecuencias sin estadística inferencial ni cruce de variables.
2. Se desea investigar la relación entre el uso de un software de auditoría y la eficiencia de los procesos en empresas mineras en 2024. Se tomarán datos una sola vez a través de cuestionarios enviados a los gerentes. Según la clasificación de Supo, este estudio es:
Respuesta correcta: a). Es observacional (no manipula variables), prospectivo (datos tomados para el estudio), transversal (una sola vez) y analítico (busca relación).
3. Un equipo desarrolla un nuevo modelo de Machine Learning que predice el comportamiento de las ventas futuras de una pyme basándose en datos históricos. epistemológicamente, ¿qué nivel de investigación y diseño corresponden?
Respuesta correcta: b). Busca predecir comportamiento futuro usando modelos (ML) con datos pasados (retrospectivo).
4. Si un estudiante quiere demostrar que su propuesta de software *mejora* la gestión de inventarios, manipulando el uso del software y midiendo los resultados en un grupo. ¿A qué nivel pertenece y cuál es la pregunta clave?
Respuesta correcta: c). Al buscar intervenir en la realidad con una propuesta (software) para solucionar un problema o mejorarlo, es nivel Aplicativo.
5. ¿Cuál es el error de diseño en un estudio que busca "Determinar las características del sistema de red actual" pero plantea un "diseño cuasi-experimental"?
Respuesta correcta: a). Caracterizar es nivel descriptivo, lo cual exige un diseño no experimental transversal. Un diseño cuasi-experimental es para el nivel explicativo.
6. ¿Qué función de la investigación se ejerce cuando un estudio establece que "el 80% de los fallos de seguridad ocurren por contraseñas débiles", sin alterar nada en el entorno?
Respuesta correcta: c). Solo está caracterizando cómo es la situación actual (estadística univariada o frecuencias).
7. La principal diferencia entre un estudio prospectivo y retrospectivo según Supo radica en:
Respuesta correcta: b). Prospectivo es recolección intencional; retrospectivo usa fuentes secundarias (bases de datos previas).
8. Un investigador extrae datos de la base de datos de la universidad (registro de notas) de los años 2020 a 2023 para evaluar la tendencia del rendimiento. ¿Qué clasificación según medición temporal y planificación le corresponde?
Respuesta correcta: b). Se miden datos en varios años (longitudinal) a partir de una base de datos secundaria ya existente (retrospectivo).
9. ¿Por qué el nivel Relacional no prueba relaciones de causalidad directa?
Respuesta correcta: a). El nivel relacional observa relaciones matemáticas/estadísticas, pero la causalidad (explicativo) requiere diseños experimentales para asegurar que A causa B.
10. Se plantea un nivel de investigación donde se requiere responder a "¿Por qué?". ¿Qué fase metodológica es crucial para este nivel que no está presente en los anteriores?
Respuesta correcta: b). Para explicar causalidad ("¿Por qué?"), se necesita un diseño experimental puro o cuasi para controlar el impacto real de las variables independientes.
11. Según los apuntes, la ciencia produce conocimiento mediante la investigación que debe ser "racional, cierto, probable... y verificado". Si en un estudio solo asumo resultados en base a mi opinión empírica sin probarlos, ¿qué fallo metodológico cometo?
Respuesta correcta: b). La ciencia requiere verificación metódica, no simples asunciones subjetivas.
12. Para implementar ciclos iterativos de la construcción de un artefacto tecnológico, ¿a qué nivel de investigación debemos apuntar?
Respuesta correcta: d). El nivel aplicativo (DSR, Design Science Research) involucra construir y validar iterativamente artefactos tecnológicos.
13. ¿Qué clasificación (Supo) define si el investigador estimó parámetros (como frecuencias y promedios) o si además probó una relación (correlación)?
Respuesta correcta: d). Lo descriptivo estima parámetros; lo analítico prueba relaciones usando variables analíticas múltiples.
14. Una investigación aplica pruebas bivariadas para cruzar la edad del usuario con el tiempo que pasa en una app. Su diseño es transversal. ¿De qué nivel estamos hablando?
Respuesta correcta: c). El uso de pruebas bivariadas para buscar relación entre dos variables sin manipulación define el nivel Relacional.
15. "Alineación lógica entre el problema, la hipótesis, las herramientas y el diseño". ¿A qué concepto de los apuntes corresponde esta definición?
Respuesta correcta: c). La coherencia epistemológica asegura que todo el planteamiento metodológico y teórico esté alineado.
16. Si tu tesis busca predecir el impacto que tendrá el abandono universitario usando series temporales de 10 años, el tipo de investigación según las mediciones es:
Respuesta correcta: c). Al usar series de tiempo (varias mediciones a través de los años), se considera longitudinal.
17. De las fases del método científico, ¿cuál es el paso lógico inmediato DESPUÉS de plantear el problema?
Respuesta correcta: d). Según las notas, las fases son: Planteamiento -> Revisión literaria -> Hipótesis -> Diseño -> Recolección -> Análisis.
18. Un sistema busca caracterizar el comportamiento del tráfico web y además estimar a qué hora habrá más saturación mañana. ¿Qué dos niveles están involucrados aquí?
Respuesta correcta: b). Caracterizar comportamiento = Descriptivo. Estimar comportamiento futuro = Predictivo.
19. ¿En qué nivel la jerarquía investigativa estipula el uso de pruebas bivariadas pero prohíbe hablar de causas directas controladas?
Respuesta correcta: d). El nivel Relacional usa pruebas bivariadas pero no prueba causa-efecto controlada (eso es del explicativo).
20. Según Supo, cuando la medición de una variable se realiza varias veces a los mismos sujetos o bases de datos, estamos ante un estudio:
Respuesta correcta: d). Longitudinal implica repetidas medidas a lo largo del tiempo.
