El problema que tiene una correcta formulación tiene que cumplir estos criterios:

1. Expresar una relación entre dos o mas variables.
2. Plantearse como pregunta interrogativa clara y sin ambigüedad
3. Admitir el contraste empírico 

> [!NOTE] Nota Importante
> Cada plantilla sintáctica en tesista la tiene instanciar para poderla utilizar 

En la formulación del problema PG ( problema general ) existen ya formulas sintácticas para los distintos niveles que existen continuación indicaremos :
## Nivel Exploratorio
En el nivel exploratorio tenemos el siguiente formula sintáctica.

$PG = ¿Como + Conector (se oresenta / se manifiesta ) + variable X + em + Ambito {A} + Periodo {T} ?$
$PEi =¿ Como + conector(" ") + variable X_i (Dimension) + en  + Ambito {A} + Periodo {T} ?$ 

## Nivel Descriptivo
Para este nivel tenemos de forma similar la formula sintáctica.
Cuando existen mas de dos variable en este caso X y Y se pude seguir dos caminos.
1. Se pude separar cada variable (Recomendable) porque cada cuenta con dimensiones complejas y indicadores que seria mejor llevarlas cada uno por su cuenta .
2. Se pude formular en una sola pero cabe señalar que cada uno estará describiendo de una población pero no se buscara una correlación entre ellas . 
[[Nivel Descriptivo (multivariable )]] Imagen Visual

$PG = ¿Cual + Conector (nivel / estado / magnitud ) + de la variable X + em + Ambito {A} + Periodo {T} ?$
$PEi =¿Cual + conector(" ") + variables descriptivas Xi + en  + Ambito {A} + Periodo {T}?$ 

## Nivel Relacional
En este nivel como el propio nombre lo indica si se busca una relación entre las variables y la formula es la siguiente : 

> [!NOTE] Formula
> PG = ¿ Que relación existe entre la variable ( X ) , y la variable ( Y ) , en el ámbito (A), periodo ( T )  ?
> PEi = ¿ Que relación existe entre la dimensiones  ( X ) , y la variable ( Y ) , en el ámbito (A), periodo ( T )  ?

## Nivel Explicativo 
[[Sesión - 1 Ciencia y Método Científico]] Definición de este nivel 
La formula sintáctica para este nivel es el siguiente  : 

> [!NOTE] Formula
> PG = ¿ En que medida la variable causa ( X ) , influye /impacta en la variable efecto ( Y ), en ambito ( A ) , periodo (T)?
>PEi =  ¿ En que medida la dimensiones  ( Xi ) , influye /impacta en la variable efecto ( Y ), en ambito ( A ) , periodo (T)?

## Nivel Predictivo
El propósito de este nivel es pronosticar el valor anticipado  de la variable  Y (Endógena) a partir de la información contenida en la variable X  (llamada exógena ).
Además el horizonte temporal  de pronostico es  (T + K ) , donde k es tiempo a futuro que pronosticaremos el valor en ese momento.
Para este nivel existen locuciones interrogativas clásicas como las siguientes: 

### Locuciones mas adecuados 
Estos son los comienzos de la pregunta por ejemplo  :  ¿ Locución + Conector ...

| Locucion                                   | Uso recomendado                         |
| ------------------------------------------ | --------------------------------------- |
| ¿Cual sera .. ?                            | pronostico global de Y ->(apartir) X    |
| ¿Que valor / nivel / tasa alcanzara ... ?  | Se desea conocer la cuantía estimada    |
| ¿En que medida predice ... ?               | Capacidad predictiva X -> Y             |
| ¿Que precision / exactitud alcanzara ... ? | Desempeño global de M con la variable X |
### Conectores mas adecuados 
Los conectores mas adecuados para este nivel son los siguientes

| Conector                                 | Finalidad                                                 |
| ---------------------------------------- | --------------------------------------------------------- |
| "pronosticada/predicha a partir de ... " | Indica Y -> deriva de X                                   |
| "... en funcion  de  ..."                | Dependencia                                               |
| " ... a partir de ... "                  | Introduce como insumo X del pronostico de Y               |
| "... que alcanzara el modelo con ..."    | Desempeño global de M con la variable X -> (pronosticar)Y |

> [!NOTE] Formula 
> PG = ¿[Pregunta] + [conector 1] +variable X + conector 2 + variable y + M (Modelo / opcional) + ambito + periodo ( T +K )?

## Nivel Aplicativo
Este nivel es donde realizamos un intervención en el problema mediante una solución ya sea software , sistema , etc. .
También el objetivo de este nivel es demostrar la eficacia de una intervención 
En este nivel también tiene sus locuciones habituales las cuales son las siguientes :
### Locuciones 
| Locucion                                      | Uso recomendado                                                                   |
| --------------------------------------------- | --------------------------------------------------------------------------------- |
| ¿En que medida influye  .. ?                  | se mide el efecto global de X sobre Y                                             |
| ¿Cómo influye ... ?                           | Cuando se busca caracterizar la dirección y la naturaleza del efecto de X sobre Y |
| ¿Cual es el efecto  ... ?                     | contrasta la magnitud del cambio del estado de Y antes de X y después de X        |
| ¿En que medida  mejora / optimiza / reduce  ? | El PG se centra en la magnitud de cambio producido de X sobre Y                   |
### Conectores 
| Conector                                        | Finalidad                                                      |
| ----------------------------------------------- | -------------------------------------------------------------- |
| "mediante la implementacion de  "               | Introduce X como la intervención que produce el efecto sobre Y |
| "... atreves de  /  … con la aplicación de ..." | Indica el instrumento mediante el cual X intervendrá en Y      |
| " ... al implementar / aplicar ... "            | Marca la implementación de X como condición de cambio en Y     |
| " ... sobre ... / ... en ... "                  | Vincula X con Y                                                |

> [!NOTE] Formula
> PG = [frase] + [conector 1] + [X: variable independiente] + [conector 2] + [variable dependiente] + [Ámbito ] + [ Periodo]
