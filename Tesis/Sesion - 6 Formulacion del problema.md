El problema que tiene una correcta formulación tiene que cumplir estos criterios:

1. Expresar una relación entre dos o mas variables.
2. Plantearse como pregunta interrogativa clara y sin ambigüedad
3. Admitir el contraste empírico 

> [!NOTE] Nota Importante
> Cada plantilla sintáctica en tesista tiene instanciar para poderla utilizar 

En la formulación del problema PG ( problema general ) existen ya formulas sintácticas para los distintos niveles que existen continuación indicaremos :
## Nivel Exploratorio
En el nivel exploratorio tenemos el siguiente formula sintáctica.

$PG = ¿Como + Conector (se oresenta / se manifiesta ) + variable X + em + Ambito {A} + Periodo {T} ?$
$PEi =¿ Como + conector(" ") + variable X_i (Dimension) + en  + Ambito {A} + Periodo {T} ?$ 

## Nivel Descriptivo
Para este nivel tenemos de forma similar la formula sintáctica.
Cuando existen mas de dos variable en este caso X y Y 
$PG = ¿Como + Conector (nivel / estado / magnitud ) + de la variable X + em + Ambito {A} + Periodo {T} ?$
$PEi =¿Como + conector(" ") + variables descriptivas Xi + en  + Ambito {A} + Periodo {T}?$
